# GOOBUS Remotion videos

Projeto Remotion para criar vídeos profissionais das páginas do site GOOBUS.

## Composições

- `Goobus-home`
- `Goobus-empresa`
- `Goobus-servicos`
- `Goobus-frota`
- `Goobus-orcamento`
- `Goobus-contato`
- `Goobus-privacidade`
- `Goobus-aluguel-de-onibus`
- `Goobus-fretamento-corporativo`
- `Goobus-turismo-excursoes`
- `Goobus-romarias`
- `Goobus-eventos`
- `Goobus-escolas-formaturas`
- `Goobus-bandas-producoes`
- `Goobus-transfers`
- `Goobus-todas-as-paginas`

## Comandos

```bash
npm run dev
npm run still:home
npm run still:frota
npm run render:showcase
```

## Padrão visual

- Formato vertical 1080x1920 para Reels, Shorts e Stories.
- Cor de destaque: `#0fb874`.
- Títulos: Plus Jakarta Sans.
- Animações controladas por `useCurrentFrame()`.
- Imagens carregadas por `staticFile()` a partir de `public/site-assets/`.
